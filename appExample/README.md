# 千行框架移动端

## 介绍
### 和千行web端相同的config配置文件，分开发环境和生产环境
### 和千行web端相同的permission配置文件
### 和千行web端相同的axios请求配置
### 移动端动态单位支持
### 使用vant-ui, 按需引入

## 安装
移动端的vuecli采用了新的稳定版vuecli5，对于node的版本有要求
你可以使用 n，nvm 或 nvm-windows 在同一台电脑中管理多个 Node 版本。
> node ^12.0.0 || >= 14.0.0

```bash
npm install
```

## 运行
```bash
npm run serve
```
